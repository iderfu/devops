## 常用命令

**显示当前正在编辑的文件名**

```
正常模式下 :f 或CTRL+G
```

**vim粘贴**

```
要粘贴时：set paste
```

**vim 将文件所有行合并到一行**

```
在 Normal Mode下执行：

ggvGJ


gg 用于跳到行首

v 转换成 visual 模式

G 跳到最后一行

J 合并行
```
